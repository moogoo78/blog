猿尾藤 [黃褥花科]

葉對生，幼葉紫紅色，基部有 2 枚腺體 (暗紅色，從野背看)

  • 細說台灣原生植物, p.178

腺果藤 [紫茉莉科]

長的像九重葛，都是紫茉莉科。紫茉莉科特徵是花萼片瓣化，筒狀，無花瓣。瘦果稜狀突出，或有腺毛。

  • 腺果藤的黏功
  • 腺果藤 @ cleyera 的部落格 :: 痞客邦 PIXNET ::
  • 福星花園: 腺果藤

風藤 [胡椒科]

幼株葉面心形 (雙重愛心)，老株卵形或長橢圓 有胡椒味

幼株在林下生長，攀附岩石或樹幹。老株爬到樹頂，地面的莖葉乾枯時，常會被認為是不同植物

  • 風藤 @ 一個人與花草的生活 :: 隨意窩 Xuite日誌

  • 風藤-杜虹花組-王彥章 - 台灣原生植物保育協會 - 我們活動與過程的記錄

  • 風藤 @ dawnchorus :: 痞客邦 PIXNET ::

  • 《野花圖鑒 2》, p.90

山棕 [棕櫚科]

賽夏族矮靈詛咒神話

  • 賽夏矮靈祭
  • 山棕 @ dawnchorus :: 痞客邦 PIXNET ::
  • 台南縣本土教學資源網 - 南瀛探索電子書
  • 國立自然科學博物館 -> 植物園 -> 花果曆 -> 回顧2011年 -> 夏天 Summer -> 山棕(8/23)

無患子 [無患子科]

科名: Sapindus 印度的肥皂。相傳無患樹的木材製成的木棒可以驅魔殺鬼，所以叫無患。

樹皮黃褐色，平滑，老樹皮很皺。在芝山岩，秋天葉子變黃。

  • 無患子
  • 特生中心 - 植物資料庫
  • 無患子-芝山文化生態綠園

龍眼 [無患子科]

葉緣反卷

  • 龍眼

軟毛柿 [柿樹科] (烏材仔)

葉中肋有毛 (小枝、葉柄也有)，枝條黑。

  • 軟毛柿
  • 軟毛柿 @ 一花一木一世界 :: 隨意窩 Xuite日誌

石苓舅 [芸香] (山橘)

單葉互生或 3~5 枚小葉的羽狀複葉，中肋表面凹下，背面隆起。腺點有香味。果實透明粉紅色，可以吃。

像橘子，橘子葉片是單身。

藥用。

  • 石苓舅

粗糠柴 [大戟科]

果實可當紅色或橙色燃料。

  • 粗糠柴

  • 《細說台灣原生植物》p.163

糙葉樹 [榆科]

去角質

老樹皮皺，縱裂脫皮。

  • 糙葉樹

朴樹 [榆科]

月桃 [薑科]

花彎曲下垂。

島田氏月桃 (葉背中肋和葉緣有毛，花、果軸直立)

柚葉藤 [天南星科]

葉子像柚樹，單身複葉。紅果。

《野花圖鑒》, p.128 《細說台灣原生植物》p.464

盤龍木 [桑科]

少數桑科木質藤本，有乳汁。

單形花，花很小，淡綠色，雄花為穗狀花序，雌花短頭狀花序，雌雄異株。
