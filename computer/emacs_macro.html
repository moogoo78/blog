強大的Emacs Macro 巨集

2013-08-30 12:06

In computer.

tags: emacs

基本操作

  1. C-x ( 開始錄製
  2. 胡搞瞎搞
  3. C-x ) 結束
  4. C-x e ABASE
  • 執行多次 C-u 次數 C-x e
  • C-x C-k n FOO 給macro取名字
  • M-x FOO 執行
  • M-x insert-kbd-macro 把macro存到設定檔, 他會問你要存那個macro name
