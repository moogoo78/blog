Lua String / 字串

2013-10-01 12:36

In computer.

tags: lua

Standard

string.find(s, pattern, [index])
string.format(s, e1, e2...)
string.sub(s, i [, j])

string.len(s)
s:len()

string.gmatch(s, pattern)
string.gsub(s, pattern, replace [, n])

string.lower(s)
string.upper(s)

string.byte("A") -- 65
string.char(65) -- A

regular expression (patterns)

.all characters
%aletters
%ccontrol characters
%ddigits
%llower case letters
%ppunctuation characters
%sspace characters
%uupper case letters
%walphanumeric characters
%xhexadecimal digits
%zthe character with representation 0

大寫表示逆向，例如: %A 就是非字串的意思

特殊字元:

( ) . % + - * ? [ ^ $

ex:

print(string.gsub("hello, up-down!", "%A", "."))
-- hello..up.down. 4
