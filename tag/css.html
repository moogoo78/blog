Performance

ab

ab -k -c 1000 -n 1000 http://testme.com

參數:

k: HTTP Keep Alive
c: concurrency (同時連線)
n: num of requests (測試的request總數, 用完就結束)
t: timelimit
p: postfile
T: content-type
h: help
