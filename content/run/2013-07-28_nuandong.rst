2013暖東峽谷山徑越野跑
#########################
:date: 2013-07-28 22:08
:categroy: run
:tags: diary


原本以為只是產業道路跑一跑，沒仔細看路線，其實山路還滿多的，

報名

但是因為跑在後段班，山徑的都變成在排隊，無法超車，要跑也跑不快，

七月底的天氣炎熱，柏油路上跑起來真的很熱，

反而原始山徑在樹蔭地下跑起來比較舒服，很多路段很陡滑也是滿挑戰的，不過常在爬山，跑起來算是一塊蛋糕阿。
